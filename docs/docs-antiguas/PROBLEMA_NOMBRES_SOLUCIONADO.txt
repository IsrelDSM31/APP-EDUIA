🔍 EL PROBLEMA ERA:

La tabla `subjects` en la base de datos usa columnas en INGLÉS:
- `name` (no `nombre`)
- `code` (no `clave`)

Pero el controlador intentaba acceder a `$subject->nombre` (español)
Por eso los nombres salían vacíos.

═══════════════════════════════════════════════════════════════════

✅ SOLUCIÓN APLICADA:

Actualicé el controlador `GradeManagementApiController.php`:

**ANTES ❌:**
```php
'subject_name' => $subject->nombre,  // nombre no existe!
```

**AHORA ✅:**
```php
'subject_name' => $subject->name,  // name es la columna correcta
```

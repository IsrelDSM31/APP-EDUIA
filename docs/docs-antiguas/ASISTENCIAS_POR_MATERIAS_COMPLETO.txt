📊 FUNCIONALIDADES POR MATERIA:
═══════════════════════════════════════════════════════════════════

✅ **REGISTRAR:** Botón "Registrar"
   → Formulario con fecha, estado, observaciones
   → Si ya existe para esa fecha → Actualiza
   → Si no existe → Crea nuevo

✅ **VER:** Botón "Ver Todo"
   → Lista completa de asistencias
   → Con botones de eliminar
   → Con botón justificar (en ausentes)

✅ **EDITAR:** Desde "Ver Todo"
   → Mismo formulario que registrar
   → Actualiza el registro existente

✅ **ELIMINAR:** Desde "Ver Todo"
   → Confirmación
   → Elimina el registro

✅ **JUSTIFICAR:** En ausencias sin justificar
   → 5 tipos de justificación
   → Campo de observaciones
   → Cambia estado a "Justificado"
